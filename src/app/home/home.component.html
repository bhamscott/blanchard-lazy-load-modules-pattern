Component Works!

The Home component and the Not Found belongs to the app-module. The Module1 and Module2 components belong to their own modules respectively. Those modules are not loaded until their routes are activated.

Lazy loading components and modules is fairly straightforward but takes some planning. Basically, the app-routing.module.ts file is where most of the magic happens:

const routes: Routes = [
{
  path: 'module1',
  loadChildren: () =>
    import('./module1/module1.module').then((m) => m.Module1Module),
    // The Module 1 component belongs to the Module 1 module!
},
{
  path: 'module2',
  loadChildren: () =>
    import('./module2/module2.module').then((m) => m.Module2Module),
    // The Module 2 component belongs to the Module 2 module!
},
{
  path: '',
  component: HomeComponent,
},
{
  path: '**',
  component: NotFoundComponent,
},
];
